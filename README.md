## 📖 Overview
This case study explores the gap between basic digital tool usage (e.g., Canva, GitHub) and true digital literacy, covering ethics, collaboration, cyber awareness, and communication. It analyzes student challenges like false confidence from templates, profile exaggeration on LinkedIn/GitHub, misuse of Google Docs, phishing risks, and poor email etiquette. The goal is to promote responsible digital citizenship through reflection and best practices.[file:1][web:18]
